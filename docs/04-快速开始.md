# Santi Channel MVP 快速开始指南

## 🚀 3分钟快速上手（极简版）

本指南将帮助你在3分钟内启动并运行 Santi Channel MVP 版本。

### 前提条件
- Node.js 18.0.0 或更高版本
- npm 包管理器
- Python 3（用于静态文件服务器）
- OpenClaw Gateway 已安装并运行

## 🛠️ 故障排除

### 常见问题

#### 1. 无法连接到 OpenClaw Gateway
```bash
# 检查OpenClaw Gateway状态
openclaw gateway status

# 如果未运行，启动它
openclaw gateway start
```

#### 2. 端口冲突
```bash
# 检查端口占用
lsof -i :3000
lsof -i :3001

# 清理占用端口
lsof -ti :3000 | xargs kill -9
lsof -ti :3001 | xargs kill -9
```

#### 3. sChat 无法访问
```bash
# 检查Python HTTP服务器
ps aux | grep "http.server 3000"

# 重启sChat
cd schat
npm start
```

#### 4. WebSocket 连接失败
```bash
# 测试WebSocket连接
node santi-channel/scripts/test-connection.js
```

## 🚢 部署到 OpenClaw

### 自动部署
```bash
cd santi-channel
node scripts/deploy.js
```

### 手动部署
1. 复制插件文件到 OpenClaw 插件目录：
   ```bash
   cp -r santi-channel ~/.openclaw/plugins/
   ```

2. 安装依赖：
   ```bash
   cd ~/.openclaw/plugins/santi-channel
   npm install
   ```

3. 重启 OpenClaw Gateway：
   ```bash
   openclaw gateway restart
   ```

## 📱 移动端访问

### 同一网络访问
如果要从手机或平板访问：
1. 确保设备在同一网络
2. 查找电脑的IP地址：
   ```bash
   ifconfig | grep "inet " | grep -v 127.0.0.1
   ```
3. 在移动设备浏览器访问：http://[电脑IP]:3000

### 注意事项
- 开发服务器默认只绑定localhost
- 生产环境需要配置CORS和网络访问
