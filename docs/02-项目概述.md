# Santi Channel 项目概述

## 🎯 项目愿景

Santi Channel 是一个为 OpenClaw AI 助手设计的自定义聊天通道插件，旨在提供类似飞书、WhatsApp 的现代化聊天体验，支持文字、文件、多媒体等多种消息类型。

## ✨ 核心特性

### 1. 消息类型全面支持
- **文字消息**: 支持 Markdown、表情符号、@提及
- **文件传输**: 图片、文档、音频、视频文件
- **多媒体消息**: 语音消息、视频消息、位置共享
- **富交互消息**: 按钮、卡片、表单、轮播图

### 2. 实时通信能力
- **低延迟**: WebSocket 实时双向通信
- **消息状态**: 发送中、已发送、已读、已回复
- **在线状态**: 用户在线/离线状态显示
- **消息同步**: 多设备消息同步

### 3. 用户与群组管理
- **用户系统**: 注册、登录、个人资料
- **联系人管理**: 添加、删除、分组联系人
- **群组聊天**: 创建、管理、退出群组
- **权限控制**: 管理员、成员、访客角色

### 4. OpenClaw 深度集成
- **无缝对接**: 直接集成 OpenClaw Gateway
- **智能路由**: 消息智能路由到相应 Agent
- **上下文保持**: 完整的对话上下文管理
- **插件扩展**: 支持自定义插件扩展
